how to create link on win and mac

2016年12月16日

最近闲下来了，就来倒腾博客，准备用我的黑果来写blog。
那么问题来了，在mac上写了一遍，win上怎么同步的问题。
不会把整个blog工程都同步吧，我确实这样干过.不过node_modules模块
文件太零碎了，这么干是不可取的。

最酷炫的一个办法是：同步source/_posts/。
下面就祭出软链接: 

ln -s /Users/jiemin/Dropbox/_posts/ /Users/jiemin/blog/source/
-------------------------------
[ln -s source target]

这条命令必须在最后加上/, 不然hexo 会报错。
