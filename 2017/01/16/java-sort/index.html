Java Sort

2017年1月16日

排序分为内部排序和外部排序，下面主要讨论内部排序。

  • 插入排序 [直接插入排序，希尔排序]
  • 选择排序 [简单选择排序，堆排序]
  • 交换排序 [冒泡排序，快速排序]
  • 归并排序
  • 分配排序

    所需辅助空间最多 归并排序
    所需辅助空间最少 堆排序
    平均速度最快 快速排序
    不稳定排序：快速排序 希尔排序 堆排序

插入排序

直接插入排序

  • 基本思想: 在一堆待排序的数中，假设前面(n-1)[n>=2]个数已经是排好序的，现在要把第n个数插到前面的有序数中，使得这个数也是排好序的。如此反复循环，指导全部排好顺序。

希尔排序(最小增量排序)

  • 基本思想: 先将要排序的一组数按某个增量d(n/2，n为要排序的个数)分成若干组,每组中记录的下标 相差d。对每组中全部元素进行直接插入排序，然后再用一个较小的增量d1(d/2)对它进行分组然后插入排序。当增量减到1时，进行直接插入排序后，排序完成。

基数排序

  • 基本思想： 将所有待比较数值统一为同样的数位长度，数位较短的数前面补零。然后，从最低位开始进行一次排序，依次进行一次排序。这样，从最低位排序一直到最高位排序完成以后，数列就变成一个有序序列。
