2017年1月11日

给pages加上小锁

  • 购买自己的域名,在godaddy上购买的.
  • 注册cloudflare来强制所有的请求都上https.

1.注册域名

付费购买一个自己想要的域名.

2.给域名添加cloudflare DNS

接下来打开cloudflare官网,注册完账号，输入你的域名(jiemin.in),按照默认设置next.
此时会让你更改goddy的域名服务器为下面的地址


更改完之后cloudflare会在全网处理，等待30min左右。直到status:Active.

此时完成了一半，下面划重点。

1.由于cloudflare接管了域名的dns,那就在dns栏设置你的A纪录,按照github官方给出的教程设置A纪录的value为

2.设置CNAME(别名)
在pages的repository里新建一个CNAME的文件[在你的blog/source/CNAME]，里面append上你的域名(jiemin.in).等一会儿在你的仓库能够看到如下的提示

之后在CloudFlare设置CNAME,如图

3.写完之后想起少了一步，需要在特殊设置一下
在page rules下：
点击create page rule


这就是我上小锁的完整记录了。零零散散的，其实我自己都看不懂。(sigh~~~)
